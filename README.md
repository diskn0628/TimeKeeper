Timekeeper iPhone 使い方ガイド
iPhoneの操作感を再現した、高精度・多機能なプロフェッショナル用タイムキーパーアプリです。
1. 基本画面構成と操作
 タブ切り替え (画面左上): * ⁠⏰ CD⁠ (カウントダウン)、⁠⏳ TM⁠ (タイマー)、⁠⏱ SW⁠ (ストップウォッチ)、⁠🧮 CALC⁠ (電卓) の4モードを切り替えます。
 時計 (画面右上):
 現在時刻を表示。長押しすると**時刻を任意の時間に手動設定（オフセット調整）**できます。
 リストの並び替え:
 各アイテムの「▲▼」ボタンをタップするか、長押ししながらドラッグすることで順番を並び替えられます。
2. 各タブの機能と使い方
⏰ CD（カウントダウン）
ターゲットとする「時刻」を指定し、そこまでの残り時間をカウントダウンします。
 追加方法: ⁠1430⁠（14:30:00）のように数字を入力して「＋」をタップ。
 プリセット: よく使う時間を「＋保存」で登録可能（長押しで削除）。
 スワイプ操作:
 左スワイプ: ラベルの編集・アイテムの削除ができます。
⏳ TM（タイマー）
設定した制限時間から「残り時間」をカウントダウンします。
 追加方法: ⁠0300⁠（3分00秒）のように数字を入力、または上のプリセットボタンから選んで「＋」をタップ。
 スワイプ操作:
 左スワイプ: ラベルの編集・アイテムの削除ができます。
 右スワイプ: **タイマー開始からの経過時間（カウントアップ）**を表示します。
⏱ SW（ストップウォッチ）
 ボタン操作:
 ⁠LAP1⁠：スプリット（分割）タイム、または前ラップからの差分タイムを記録。
 ⁠LAP2⁠：単純な累積ラップタイムを記録。
 ⁠RST⁠：全記録をリセット。
 表示モード切り替え (右上「#1 / 1→2」ボタン):
 タップで通常モード（各ラップ単体の時間）と、差分モード（前のラップよりどれだけ遅い/早いか）を切り替えられます。
🧮 CALC（時間電卓）
時間の足し算・引き算を素早く行えます。
 入力例: ⁠1030⁠（10分30秒） ⁠+⁠ ⁠50s⁠（50秒） ⁠+⁠ ⁠2m⁠（2分）
 コピー: ⁠📋コピー⁠をタップすると、結果（⁠hhmmss⁠形式の6桁数字）をクリップボードにコピーします。コピーした値は他のタブの入力欄に自動反映されます。
3. プロ仕様の便利機能
⚡ 先行タイムアップ（−◯秒ボタン）
 タップで有効化。設定した秒数（例：2秒）だけ**手前でタイムアップ（残り0秒）**を告げるようにします。
 ボタンを長押しすると、早める秒数を自由に変更できます（1〜999秒）。
↗ フローティング小窓化 ＆ 外部ディスプレイ同期
 リスト上のアイテムや時計をダブルタップすると、画面上にドラッグ＆リサイズ可能な「フローティング小窓」が出現します。
 小窓の右上にある ⁠↗⁠ ボタンをタップすると、別ブラウザウィンドウとして同期画面をポップアップ表示できます。これを外部モニター等に配置することで、プレゼンや配信用ディスプレイとして活用可能です。
(※開かない場合は、ブラウザの設定でポップアップを許可してください)
