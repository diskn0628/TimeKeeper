Timekeeper iPhone 
ユーザーズガイド
Timekeeper iPhoneは、放送現場や時間管理がシビアな業務向けに設計された、オフライン対応の高精度マルチ時間管理アプリです。「カウントダウン(CD)」「タイマー(TM)」「ストップウォッチ(SW)」「時間電卓(CALC)」の4つの機能をシームレスに切り替えて使用できます。
1. 共通ヘッダー機能
画面最上部には、ナビゲーションタブと現在時刻を管理する共通ヘッダーが常駐しています。
現在時刻とオフセット（時計エリア）

 時計の自動同期: 起動時にローカルデバイスの時計と自動的に同期します。

 時刻の手動設定（長押し）: 1. 時計エリアを約0.6秒長押しすると、時刻入力モーダルが表示されます。
2. 数値を「⁠143000⁠（14時30分00秒）」のように6桁（hhmmss）で入力し、「✓ 設定」をタップします。
3. 設定すると、現在時刻の右に「⁠★⁠」マークが点灯し、デバイスの実際の時刻からオフセット（ズレ）を保ったまま時計が進行します。
4. リセットしたい場合は、再度長押しして「RST」をタップします。
2. 各タブ機能の使い方
⏰ CD（カウントダウン）タブ
特定の「目標時刻」に向けてカウントダウンを行う機能です。

 時刻の追加方法:

 入力欄に数字4桁（例: ⁠1430⁠ → 14時30分00秒）または6桁（例: ⁠143045⁠ → 14時30分45秒）を入力し、「＋」ボタンをタップします。

 すでに過ぎた時刻を入力した場合、自動的に「翌日のその時刻」としてカウントダウンを計算します。

 オプションで「ラベル」を入力しておくと、目標値に名前（例: 「本番」「コーナー終了」など）を付けることができます。

 プリセットの管理:

 よく使う時刻を入力した状態で「＋ 保存」ボタンをタップすると、ボタンとして最上部に保存されます。

 保存されたプリセットをタップするだけで、ワンタップで新規カウントダウンを追加できます。

 プリセットボタンを約0.6秒長押しすると、そのプリセットを削除できます。

 状況に応じた色変化:

 残り時間1分未満：琥珀色（アンバー）に変化。

 目標時刻超過（タイムアップ）：赤色になり、経過時間を「+」表記でミリ秒加算します。
⏳ TM（タイマー）タブ
「設定された時間」からゼロに向けてカウントダウンするマルチタイマー機能です。複数のタイマーを完全に独立して同時に並行稼働させることができます。

 タイマーの追加方法:

 入力欄に「分」「秒」を直接入力して追加できます。

 末尾に「⁠m⁠」を付けると分指定（例: ⁠5m⁠ → 5分）、末尾に「⁠s⁠」を付けると秒指定（例: ⁠90s⁠ → 1分30秒）として直接パースされます。

 ⁠0300⁠（3分）のように4桁の数字を入力して追加することも可能です。

 プリセットテンプレート:

 最上部に「1分」「2分」「3分」「5分」などのデフォルトテンプレートが配置されています。

 テンプレートをタップすると、ボタンの枠線がネオンブルーに美しく発光し、現在どのテンプレートが選択されているかが視覚的に分かります。

 テンプレートボタンを約0.6秒長押しすると、時間の変更や名称の書き換え、削除が可能です。

 操作ボタン:

 ⁠▶開始⁠ / ⁠▶再開⁠ / ⁠⏸停止⁠: 各タイマーカードの右側で操作します。

 ⁠↺⁠: タイマーの設定初期値（トータル時間）にリセットします。
⏱ SW（ストップウォッチ）タブ
ミリ秒（100分の1秒精度）で計測可能な高機能ストップウォッチです。一般的な計測に加え、独自の「LAP1（差分/分割）」「LAP2（累積ラップ差）」のダブルリスト計測が可能です。

 基本操作:

 ⁠▶ スタート⁠ / ⁠⏸ 停止⁠: 中央のメインボタンで計測を開始・停止します。

 ⁠RST⁠: 停止中に左上の「RST」をタップすると、計測値とラップ履歴がすべてリセットされます。

 「1→2（差分）モード」の切り替え:

 右上の「1→2」ボタンをタップすると、ラップリストの表示モードを「標準（#1）」と「差分（1→2）」で切り替えることができます。

 ダブルラップリストの仕様:

 LAP1（左側リスト）:

 ストップウォッチ稼働中に左下の ⁠LAP1⁠ をタップして記録します。

 各ラップ間の区間タイム（スプリット）と、直前のラップからの相対的な差分時間（⁠+秒数⁠）を表示します。

 記録の中で「最も速いラップ（緑色 ▲）」と「最も遅いラップ（赤色 ▼）」が自動でマークされます。

 LAP2（右側リスト）:

 ストップウォッチ稼働中に右下の ⁠LAP2⁠ をタップして記録します。

 純粋なトータルタイムの累積値を記録し、ラップごとの累積変化を追うことができます。
🧮 CALC（時間電卓）タブ
「14時30分15秒 ＋ 45分 ＝ ？」といった、時間・時刻の計算を即座に行える専用の電卓機能です。

 入力ルール:

 入力された数値は、hhmmssとして解釈されます。

 数字のあとに ⁠s⁠（秒）や ⁠m⁠（分）を入力して直接加減算できます（例: ⁠143000⁠ ⁠+⁠ ⁠15m⁠ ⁠-⁠ ⁠30s⁠）。

 クリップボード連携:

 計算結果の右側にある「📋 コピー」ボタンをタップすると、結果が ⁠143000⁠ のような6桁の形式でデバイスのクリップボードにコピーされます。

 このコピーされた値は、他のタブ（CD、TM、SW）を開いた際、自動的に新規追加用の入力欄にペーストされます。
3. スマートフォンのための特殊ジェスチャー（極めて重要）
Timekeeper iPhoneには、画面上のパーツを削減し表示領域を最大化するため、直感的で高度なタッチジェスチャーが多数組み込まれています。
① 長押し並び替え（CD・TMタブ）

 操作方法:

 カウントダウン（CD）やタイマー（TM）のリスト内で、任意のアイテムを約0.4秒間長押しします。

 検知されると、デバイスがわずかに振動（バイブレーション）し、対象のカードが琥珀色の枠に発光して少し浮き上がります。

 そのまま指を離さずに上下にスライドさせることで、狙った場所に直感的に並び替えることができます。

 タッチスクロールとの完璧な共存:

 長押しせずに画面を上下にフリックすると、長押し判定が即座にキャンセルされ、引っかかりのない滑らかな慣性スクロールに切り替わります。
② スワイプ削除（CD・TM・SWタブ）

 操作方法:

 リスト内のアイテムを左方向へ水平にスワイプします。

 スワイプした距離に応じて右から赤い「🗑 削除」ボタンが引き出されます。

 スワイプを止めると削除ボタンがホールドされ、タップすることで対象のアイテムを安全に削除できます。元に戻したい場合は右にスワイプして閉じます。
③ ダブルタップ小窓化（CD・TM・SWタブ）

 操作方法:

 稼働中の「タイマー」「カウントダウン」「ストップウォッチ」のカードを**すばやくダブルタップ（0.35秒以内に2回タップ）**します。

 すると、アプリの上に「ドラッグ移動・リサイズ可能」な**フローティングミニウィンドウ（ピクチャー・イン・ピクチャー風の小窓）**が生成されます。

 小窓の操作:

 移動: 小窓の任意の場所を掴んで画面上のどこにでも配置できます。

 サイズ変更: 小窓の右下角にある三角マークの「リサイズハンドル」をドラッグすると、自由な縦横比にサイズ変更でき、文字の大きさもウィンドウサイズに自動追従して最大化します。

 上下バランス調整: 小窓の中央にある境界線（スライダーバー）を上下にドラッグすることで、上段の「ラベル」と下段の「タイム数字」の表示バランス（フォントサイズ）をカスタマイズできます。

 非表示 / 閉じる: 小窓をタップすると右上に「✕」ボタンが出現し、タップすると小窓が消えて元のリストに戻ります。
④ 時間盗み（秒盗み）機能（CD・TMタブ）

 概要:

 目標時刻やタイマーを、設定した「n秒前」にフライングでタイムアップ（アラート状態）にさせる放送・タイムキーピングに必須の機能です。

 設定方法:

 タブ上部にある「⁠−0秒⁠」ボタンをタップすると、機能が有効化（琥珀色に点灯）します。

 ボタンを約0.5秒長押しすると設定モーダルが開き、何秒前（例: 2秒、5秒など）にするかを自由に入力・カスタマイズできます。
