Timekeeper iPhone 使い方ガイド
1. 基本操作（PC・スマホ共通）
 項目をスワイプ（ドラッグ）: * 左へ: ラベル編集・削除ボタンが出現
 右へ（タイマーのみ）: 稼働中の「経過時間」を裏表示
 項目を長押し: ドラッグで上下に並び替え
 項目をダブルタップ: フローティング小窓化
2. 現在時刻（画面右上）の操作
 ダブルタップ: 横画面の「全画面置時計モード」に切り替え
 横画面から戻る時も画面をダブルタップします。
 長押し（0.6秒）: 時刻の手動設定（オフセット調整）
3. モード別ガイド
⏰ CD (カウントダウン時刻)
 指定時刻（例 ⁠1430⁠ = 14:30）までの残り時間を計測。
 「＋保存」でよく使う時間をプリセット保存（長押しで削除）。
⏳ TM (タイマー)
 制限時間（例 ⁠0300⁠ = 3分）からのカウントダウン。
 複数のタイマーを同時に並行してスタート・一時停止可能。
⏱ SW (ストップウォッチ)
 LAP1: スプリットタイム（分割時間または差分）を記録。
 LAP2: 単純な累積ラップタイムを記録。
 #1 / 1→2 ボタン: ラップごとの差分（比較）モードに切り替え。
🧮 CALC (時間電卓)
 ⁠10m⁠ (10分) ⁠+⁠ ⁠30s⁠ (30秒) などの時間計算。
 「コピー」すると他タブの入力欄に自動反映。
4. 便利機能
 「−○秒」ボタン: 設定秒数だけ手前でタイムアップさせます（長押しで秒数設定）。
 小窓の「↗」ボタン: 外部ディスプレイ用の別ウィンドウを起動（※要ポップアップ許可）。
